# 🚀 LifeOS

LifeOS is a personal life tracking system built in Python.

## Features (Planned)

- 📚 Study Tracker
- 💪 Workout Tracker
- 🍽 Meal Tracker
- 💧 Water Intake
- 💤 Sleep Tracker
- 🐕 Dog Walk Tracker
- 😊 Mood Tracker
- 📊 Daily Reports
- 📈 Analytics Dashboard

## Tech Stack

- Python
- JSON (Phase 1)
- SQLite (Phase 2)
- Pandas
- Matplotlib
- Power BI
