# 🚀 LifeOS

LifeOS is a personal life tracking system built with Python.

The goal of the project is to record daily activities such as study sessions, workouts, meals, sleep, and walks, then analyze the data over time to improve productivity, fitness, and consistency.

---

## ✨ Features

### Current (Phase 1)

- 📚 Study Tracking
- 💪 Workout Tracking
- 🍽️ Meal Tracking
- 💤 Sleep Tracking
- 🚶 Walk Tracking
- 📄 Daily JSON Logs

### Planned

- 📊 Daily Report Generation
- 📅 Weekly Reports
- 📆 Monthly Reports
- 📈 Analytics with Pandas
- 📉 Data Visualization with Matplotlib
- 🗄️ SQLite Database Migration
- 📊 Power BI Dashboard

## 🎯 Project Goals

- Build a real-world Python project from scratch.
- Learn software architecture and modular programming.
- Track personal habits using structured JSON data.
- Analyze long-term trends using data analytics tools.
- Gradually evolve from JSON to SQLite without redesigning the application.
